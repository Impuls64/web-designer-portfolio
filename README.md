# Портфолио — WEB Дизайнер (Отклик на вакансию)

Папка подготовлена для отправки заказчику. Все материалы готовы к использованию.

## Структура

| Файл | Описание |
|------|----------|
| `cover-letter.md` | Сопроводительное письмо — отправляй первым сообщением |
| `project-01-figma-tilda.md` | Кейс: лендинг в Figma + сборка в Tilda |
| `project-02-figma-wordpress-ai.md` | Кейс: корпоративный сайт в Figma + WordPress + нейросети |
| `project-03-automation-ai.md` | Кейс: автоматизация дизайн-процессов через AI |

## Что заменить перед отправкой

1. **Контакты** в `cover-letter.md` (Telegram, Email, портфолио)
2. **Ссылки на Figma** в кейсах (сейчас заглушки)
3. **Контакты** в `project-03-automation-ai.md`
4. При необходимости — адаптируй тексты под свой стиль и реальные проекты

## Формат работы

Заказчик ищет партнера в долгую. В письме акцент на:
- Самостоятельность и ответственность
- Опыт с нейросетями (Midjourney, ChatGPT, Stable Diffusion)
- Автоматизацию и вайбкодинг
- Готовность к тестовому проекту
