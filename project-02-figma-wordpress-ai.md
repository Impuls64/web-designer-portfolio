# Кейс: Корпоративный сайт для IT-консалтинга

## Проект

**Название:** DataFlow Consulting — корпоративный сайт  
**Клиент:** ООО «ДатаФлоу Консалтинг» (Москва, РФ)  
**Сфера:** B2B IT-консалтинг, внедрение CRM и автоматизация бизнес-процессов  
**Роль:** Веб-дизайнер + AI-интегратор (соло-проект)  
**Срок:** 6 недель  
**Статус:** Выпущен в продакшен

---

## Задача

Клиент — компания из 45 человек, которая внедряет amoCRM, Битрикс24 и собственные решения на базе Python/FastAPI для среднего бизнеса. У них был устаревший сайт на конструкторе 2018 года: неадаптивный, без CMS, с фотостоковыми изображениями.

**Цели:**
- Повысить доверие среди корпоративных заказчиков (средний чек проекта — 800K–2M ₽)
- Увеличить конверсию в заявку с 0.4% до 2%+
- Автоматизировать первичную квалификацию лидов
- Создать контент без привлечения копирайтера и разработчика

**Ограничения:**
- Бюджет без строки «разработка»
- Нужна CMS, которую клиент сможет редактировать сам
- Срок — до конференции «CRM-Экспо» через 6 недель

## Дизайн в Figma

### Компонентная система

Создал полноценный дизайн-систему в Figma на 180+ компонентов:

- **Цветовая палитра:** 12 семантических токенов (primary, accent, surface, text-default и др.) в переменных Figma
- **Типографика:** Inter + JetBrains Mono (для блоков кода и технических кейсов), 6 уровней заголовков, scale 1.25
- **Сетка:** 12-колоночная, gap 24px, max-width 1280px
- **Брейкпоинты:** Desktop (1440+), Tablet (768–1439), Mobile (<768)

**Ключевые компоненты:**
- `Button` — 3 варианта (primary, secondary, ghost), 2 размера, состояния default/hover/active/disabled/loading
- `Card/Service` — карточка услуги с иконкой, заголовком, описанием и CTA
- `Section/CTA` — универсальный блок призыва к действию
- `Form/Lead` — форма захвата лида, интегрированная с amoCRM
- `Nav/Desktop` и `Nav/Mobile` — адаптивная навигация

### UI-kit

Собрал отдельный файл `dataflow-ui-kit.fig` с:
- Всеми цветовыми токенами и типографическими стилями
- Иконочной библиотекой (240 иконок, оптимизированных в SVG)
- Примерами состояний интерактивных элементов
- Чеклистом доступности (контрастность 4.5:1+, focus-стили)

### Прототипирование

- **Интерактивный прототип** на 24 экрана с переходами и микроанимациями
- **User flow** для 3 сценариев: «Узнать об услуге», «Оставить заявку», «Скачать кейс"
- **Тестирование** провел через Maze.co — 12 респондентов, выявил 2 проблемы с навигацией на мобильном, исправил за 2 часа

### AI в дизайне

- **Midjourney v6:** Сгенерировал 8 hero-изображений для главных разделов. Промпт:  
  `Corporate technology consulting, abstract data flows, dark blue and electric cyan gradient, minimal, clean, professional, no text --ar 16:9 --style raw --v 6`  
  Из 8 вариантов выбрал 3, доработал в Photoshop (убрал артефакты, добавил корпоративные цвета).
- **Figma AI (beta):** Использовал для первичного naming слоев и генерации вариантов layout для страницы «Кейсы». Сократил время на разведку на 40%.
- **Relume AI:** Сгенерировал wireframe-структуру для страницы «Услуги» по текстовому описанию, затем кастомизировал под бренд.

## Сборка в WordPress

### Выбор инструментов

Решил собрать без разработчика, используя:
- **WordPress 6.4** + **Kadence Blocks Pro** (библиотека кастомных блоков)
- **Kadence Theme** (легкая тема с полным контролем дизайна)
- **ACF PRO** — кастомные поля для кейсов и отзывов
- **WPForms** + интеграция с amoCRM через webhook
- **Yoast SEO Premium**

**Почему не Elementor:** Kadence Blocks легче (загрузка страницы 0.8s против 2.4s у Elementor), лучше совместим с Gutenberg, проще передавать клиенту.

### Реализация дизайна

- **Все секции** сверстал через Kadence Blocks (Row Layout, Advanced Button, Info Box, Testimonial, Post Grid)
- **Кастомные блоки ACF:** Карточка кейса, Таймлайн процесса, Калькулятор ROI
- **Адаптив:** Настроил поведение сеток под каждый брейкпоинт через встроенные responsive-настройки Kadence
- **Производительность:** 
  - Lazy loading для изображений
  - WebP-формат (сжал через Squoosh.ai)
  - Критический CSS встроен в head
  - Итог: 94/100 mobile, 99/100 desktop (PageSpeed Insights)

### Интеграции

- **amoCRM:** Формы захвата отправляют лида сразу в воронку, проставляют теги и ответственного
- **Яндекс.Метрика + Google Analytics 4:** Цели на все микроконверсии
- **Calendly:** Встроил виджет бронирования встреч

## Использование нейросетей и AI-инструментов

### Генерация контента (ChatGPT-4 + Claude)

**Задача:** Нужно было 25 000+ знаков продающего текста без копирайтера.

**Процесс:**
1. **Интервью с клиентом** записал, расшифровал в Whisper API
2. **ChatGPT-4:** Создал структуру сайта, UTP, тексты для 8 ключевых страниц  
   Промпт: *«Ты senior B2B копирайтер. Напиши текст для страницы внедрения amoCRM для среднего бизнеса. Тон: экспертный, но без воды. Акцент на ROI и автоматизации. Объем — 800 знаков.»*
3. **Claude 3 Sonnet:** Провел ревизию — убрал канцеляризмы, добавил конкретику (цифры, сроки)
4. **Итог:** Редактура заняла 4 часа вместо 20+ на написание с нуля

**Что автоматизировал:**
- Meta-теги для всех страниц (Yoast + ChatGPT)
- Alt-тексты для 34 изображений
- Тексты для CTA-кнопок (A/B-варианты)
- Структурированные описания для Schema.org разметки

### Генерация изображений (Midjourney + Photoshop AI)

- **Hero-баннеры:** 6 изображений, сгенерированных в Midjourney
- **Иконки и иллюстрации:** Часть иконок — из Phosphor Icons, кастомные иллюстрации для «Процесса работы» — сгенерированы в Midjourney + доработаны в Photoshop (Generative Fill для удаления артефактов)
- **Фото команды:** Реальные фото прошли через Retouch4me + Photoshop Neural Filters для единообразия цветокоррекции

### SEO-оптимизация через AI

- **SurferSEO:** Провел анализ топ-10 по 12 целевым запросам, получил рекомендации по структуре и ключевым словам
- **ChatGPT:** Сгенерировал FAQ-раздел на основе People Also Ask из Google (спарсил через ZenRows)
- **Screaming Frog + ChatGPT:** Нашел 23 дублирующих meta-description, исправил пакетно через CSV
- **Результат:** Через 3 месяца после запуска — 14 запросов в топ-10 Яндекса, 8 в Google

### Автоматизация процессов

- **n8n:** Настроил воркфлоу:
  - Новая заявка с сайта → amoCRM + уведомление в Telegram-чат отдела продаж
  - Скачивание кейса → добавление email в Mailchimp-сегмент
  - Запись на встречу → Google Calendar + Zoom-ссылка
- **ChatGPT API:** Интегрировал чат-бота на первом экране (виджет через Tidio), который квалифицирует лида перед передачей менеджеру. Бот отвечает на 70% вопросов без человека.

## Вайбкодинг элементов

Несмотря на «no-code» подход, потребовались 2 кастомные вставки:

### 1. Калькулятор ROI

Клиент хотел интерактивный калькулятор экономии от внедрения CRM. Kadence Blocks такого не предоставляет.

**Решение:** Сгенерировал HTML/CSS/JS через ChatGPT-4 (Code Interpreter), доработал под дизайн-систему.

```html
<!-- Вставка через Custom HTML блок Gutenberg -->
<div id="roi-calculator" class="df-roi-calc">
  <div class="df-input-group">
    <label>Сотрудников в отделе продаж</label>
    <input type="range" id="employees" min="1" max="50" value="10">
    <span class="df-value">10</span>
  </div>
  <div class="df-input-group">
    <label>Средняя зарплата, ₽</label>
    <input type="number" id="salary" value="80000">
  </div>
  <div class="df-result">
    <p>Экономия в год: <strong id="savings">0 ₽</strong></p>
    <p class="df-note">*Расчет при автоматизации 30% рутинных задач</p>
  </div>
</div>
```

CSS и JS — также сгенерированы AI, затем протестированы вручную. Время на реализацию: 2 часа вместо 8+ у разработчика.

### 2. Анимация счетчиков

Анимированные цифры в блоке «Результаты в цифрах» (150+ проектов, 45 клиентов, 98% NPS).

**Решение:** Использовал анимацию из библиотеки CountUp.js, подключил через Code Snippets плагин. Параметры анимации (duration, easing) подобрал через перебор с AI-рекомендациями.

### 3. Динамический блок «Последние кейсы"

Нужен was фильтр по отраслям ( Retail, IT, Производство) без перезагрузки.

**Решение:** Сгенерировал jQuery-скрипт через Claude, встроил через Kadence Elements Hook. Работает из коробки WordPress + AJAX.

## Ключевые решения

### 1. AI-first подход к контенту
Вместо «сначала дизайн, потом тексты» работал параллельно. GPT-4 генерировал тексты на основе интервью, я сразу проверял их вписывание в сетку. Это сократило циклы правок с 5 до 2.

### 2. Компонентная система как единый источник правды
Все 180+ компонентов были согласованы с клиентом еще до перехода к WordPress. Клиент видел точную копию будущего сайта в Figma — никаких сюрпризов при сборке.

### 3. No-code + вайбкодинг вместо разработки
Каденс покрыл 95% потребностей. Оставшиеся 5% (калькулятор, фильтры, анимации) реализовал через AI-генерацию кода без найма разработчика. Экономия бюджета клиента — ~200K ₽.

### 4. Автоматизация на уровне бизнес-процессов
Сайт — не брошюра, а часть воронки продаж. Интеграция с amoCRM, n8n-воркфлоу и чат-бот позволили клиенту обрабатывать в 3 раза больше лидов той же командой.

### 5. Производительность как фича
В B2B IT-консалтинге аудитория — CTO и IT-директора, которые ценят скорость. 0.8s загрузки и 99 баллов PageSpeed — это сигнал компетентности.
